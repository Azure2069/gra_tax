# BizInvoice Ghana lib scaffold

Copy the `lib` folder into your Flutter project.

Add these dependencies to `pubspec.yaml`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
  go_router: ^16.0.0
  url_launcher: ^6.3.0
```

This starter uses in-memory repositories so it can be understood and replaced gradually with Drift or sqflite. Tax values are placeholders and must be verified before production use.
